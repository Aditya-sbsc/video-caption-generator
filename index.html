Help & Documentation

Quick Start Guide

  1. Live Captions: Click "Start Recording" to generate real-time captions from your microphone
  2. Video Upload: Drag and drop a video file to extract captions from the audio
  3. Caption Editor: Use the timeline to edit, split, merge, and modify captions
  4. Styling: Customize fonts, colors, and effects for professional-looking captions
  5. Export: Download captions in SRT, VTT, ASS, or JSON formats

Keyboard Shortcuts

  • Space - Play/Pause timeline
  • Ctrl + Z - Undo
  • Ctrl + Y - Redo
  • Delete - Delete selected caption
  • Ctrl + S - Save project
  • Ctrl + E - Export captions

Supported Languages

Speech recognition supports 50+ languages including English, Spanish, French, German, Italian, Portuguese, Russian, Japanese, Korean, Chinese, Arabic, and Hindi.

File Format Support

Input: MP4, WebM, MOV, AVI (up to 500MB)

Export: SRT, VTT, ASS, JSON formats
